# Решение проблемы с предупреждением безопасности браузера

## Проблема
Браузер Chrome показывает предупреждение "Опасный сайт" для домена `giftdraw.today`.

## Возможные причины:

1. **Домен не настроен в Vercel или не имеет валидного SSL сертификата**
2. **Домен был помечен Google Safe Browsing как опасный**
3. **Проблемы с DNS настройками**

## Решение:

### Шаг 1: Проверьте настройки домена в Vercel

1. Откройте [Vercel Dashboard](https://vercel.com/dashboard)
2. Выберите проект `crypto-lottery-today`
3. Перейдите в **Settings** → **Domains**
4. Убедитесь, что домен `giftdraw.today` добавлен и имеет статус **"Valid"** (зеленая галочка)
5. Если домена нет или статус не "Valid":
   - Нажмите **Add Domain**
   - Введите `giftdraw.today`
   - Следуйте инструкциям для настройки DNS записей

### Шаг 2: Проверьте DNS записи

Убедитесь, что DNS записи настроены правильно у вашего регистратора домена:

**A запись:**
- Type: A
- Name: @ (или оставьте пустым)
- Value: IP адрес от Vercel (обычно `76.76.21.21`)

**ИЛИ CNAME запись:**
- Type: CNAME
- Name: @
- Value: `cname.vercel-dns.com`

### Шаг 3: Проверьте SSL сертификат

1. В Vercel Dashboard → Settings → Domains
2. Убедитесь, что для домена `giftdraw.today` есть SSL сертификат
3. Vercel автоматически выдает SSL через Let's Encrypt
4. Если сертификата нет, подождите 5-60 минут после добавления домена

### Шаг 4: Проверьте статус домена в Google Safe Browsing

1. Откройте [Google Safe Browsing Site Status](https://transparencyreport.google.com/safe-browsing/search?url=giftdraw.today)
2. Введите `giftdraw.today`
3. Если домен помечен как опасный:
   - Подайте запрос на пересмотр в Google Search Console
   - Это может занять несколько дней

### Шаг 5: Очистите кэш браузера

1. Откройте Chrome → Настройки → Конфиденциальность и безопасность
2. Нажмите "Очистить данные просмотра"
3. Выберите "Кэшированные изображения и файлы"
4. Нажмите "Очистить данные"

### Шаг 6: Проверьте домен через инструменты

1. [SSL Labs SSL Test](https://www.ssllabs.com/ssltest/) - проверка SSL сертификата
2. [What's My DNS](https://www.whatsmydns.net/) - проверка DNS записей
3. [Google Transparency Report](https://transparencyreport.google.com/safe-browsing/search) - проверка статуса в Safe Browsing

## Временное решение

Если домен еще не настроен в Vercel, используйте временно домен Vercel:

1. Найдите ваш Vercel домен (например, `crypto-lottery-today.vercel.app`)
2. Используйте его для тестирования
3. После настройки кастомного домена переключитесь на `giftdraw.today`

## Важные замечания

- **Время распространения DNS**: Обычно 5-60 минут, но может занять до 24 часов
- **SSL сертификат**: Vercel автоматически выдает SSL, но это может занять время
- **Google Safe Browsing**: Если домен новый, он может быть помечен как подозрительный до проверки

## Если проблема сохраняется

1. Убедитесь, что домен правильно настроен в Vercel
2. Проверьте, что DNS записи указывают на Vercel
3. Дождитесь полного распространения DNS (до 24 часов)
4. Проверьте статус в Google Safe Browsing
5. Обратитесь в поддержку Vercel, если проблема не решается
